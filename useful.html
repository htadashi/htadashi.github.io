Recommended open-source softwares

  • Octave - Free alternative to MATLAB, with a language that is mostly compatible.

  • Mathics - Free alternative to Mathematica, with a language that is mostly compatible.

  • Semidefinite programming in Python - A collection of Python libraries that can be used to solve LMIs and other optimization problems.

  • Python-Control - Python module that implements basic operations for analysis and design of feedback control systems.

  • Harold - Another open-source systems and controls toolbox for Python3.
