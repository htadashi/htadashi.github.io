Useful resources

👨‍💻 Recommended open-source softwares

  • Octave - Free alternative to MATLAB, with a language that is mostly compatible.

  • Mathics - Free alternative to Mathematica, with a language that is mostly compatible.

  • http://matheclipse.org/ - computer algebra language and Java symbolic math library.

  • Semidefinite programming in Python - A collection of Python libraries that can be used to solve LMIs and other optimization problems.

  • Python-Control - Python module that implements basic operations for analysis and design of feedback control systems.

  • Harold - Another open-source systems and controls toolbox for Python3.

  • https://github.com/JuliaControl/ControlSystems.jl - Control system toolbox for https:julialang.org/.

  • GurobiLink - Gurobi interface with Wolfram Language.

🏫 Didactic and drawing resources

🚴 Control theory

  • https://resourcium.org/ - A collection of control and automation resources.

  • http://janismac.github.io/ControlChallenges/ - Simulate some control algorithms to different control problems in Javascript.

  • Interactive course for control theory - Course with interactive Jupyter notebooks.

  • https://lpsa.swarthmore.edu/Animations/ - Javascript tool for animation of linear physical systems

  • https://github.com/Nukleon84/pyflowsheet - Python package to create process flow diagrams (PFD)

  • http://www.nonlinearbenchmark.org/index.html - Database of systems for benchmarking nonlinear system identification algorithms.

  • https://cheever.domains.swarthmore.edu/ - Interactive demos for teaching classical control of linear systems.

  • https://demonstrations.wolfram.com/topic.html?topic=Control+Theory&limit=20 - Mathematica demonstrations related to control theory.

  • https://unilabs.dia.uned.es/ - A network formed by many universities that share their laboratory resources for education purposes.

🤖 Robotics

  • Robotarium project - A remotely accessible swarm robotics research platform that remains freely accessible to anyone.

  • OFFWORLD GYM - Open-access physical robotics environment for real-world reinforcement learning benchmarking and research.

  • roboton.io - Roboton.io is a virtual robot-competition platform that can be used entirely from the browser.

🔋Electronics

  • https://www.withdiode.com/ - 3D circuit simulator. Can be used to program Arduino boards.

  • https://www.falstad.com/circuit/ - Online analog circuit simulator

  • simcirjs - Digital circuit simulator in HTML5 and Javascript

  • Yosys2digitaljs - Online SystemVerilog synthetizer

  • SKiDL - SKiDL is a module that allows you to compactly describe the interconnection of electronic circuits and components using Python.

  • Wireviz - WireViz is a tool for easily documenting cables, wiring harnesses and connector pinouts. It takes plain text, YAML-formatted files as input and produces beautiful graphical output

🧰 Miscellaneous

  • nbinteract - Generate Interactive Web Pages From Jupyter Notebooks. See also this tutorial.

  • RISE - Turn Jupyter notebooks into slideshow

  • ObservableHQ - Interactive notebooks with Javascript. For an interesting example, see notebook(https:observablehq.com@patrickmineaultstabilizing-the-cart-pole-inverse-pendulum-using-lqr).

  • Jupyter{book} - Build books with Jupyter

  • MathBox - Presentation-quality WebGL math graphing (see also How to use Jupyter with MathBox2 plots)

  • EulerTool - Online editor for Manim, an animation engine for explanatory math videos

  • Jupyter-Manim - Jupyter notebook magic to integrate manim within output cells.

  • Math Inspector - A visual programming environment for scientific computing.

  • CindyJS - A framework to create interactive (mathematical) content for the web.

  • SageMathCell - SageMathCell can be used to embed Sage computations into any webpage.

  • Wolfram Demonstrations Project - Database of interactive demonstrations made with Mathematica.

  • MATLAB Grader - Automatically grade MATLAB code in any learning environment

  • For recording and creating online courses:

    • https://www.grapic.co/ - Collaborate on real whiteboards and notebooks in remote and hybrid meetings.

    • https://sanas.ai/ - Allow speakers to speak in different accents

    • https://www.descript.com/ - Edit audio by editing text. (English only)

    • https://www.narakeet.com/ - Use markdown syntax to create video clips from images, add captions, and life-like narration audio from your text. (Has Portuguese narration)

    • https://www.synthesia.io/ - Automatically redub videos (and even edit the mouth movements) to other languages with A.I.
