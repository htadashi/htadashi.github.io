Useful links

Recommended open-source softwares

  • Octave - Free alternative to MATLAB, with a language that is mostly compatible.

  • Mathics - Free alternative to Mathematica, with a language that is mostly compatible.

  • http://matheclipse.org/ - computer algebra language and Java symbolic math library.

  • Semidefinite programming in Python - A collection of Python libraries that can be used to solve LMIs and other optimization problems.

  • Python-Control - Python module that implements basic operations for analysis and design of feedback control systems.

  • Harold - Another open-source systems and controls toolbox for Python3.

  • https://github.com/JuliaControl/ControlSystems.jl - Control system toolbox for https:julialang.org/.

Didactic and drawing resources:

Control theory

  • http://janismac.github.io/ControlChallenges/ - Simulate some control algorithms to different control problems in Javascript.

  • https://lpsa.swarthmore.edu/Animations/ Javascript tool for animation of linear physical systems

  • https://unilabs.dia.uned.es/ - A network formed by many universities that share their laboratory resources for education purposes.

    • https://www.youtube.com/user/ControlLectures - Brian Douglas channel

  • http://controlcartoons.com/ - A cartoon tour of control theory

Robotics

  • Robotarium project - A remotely accessible swarm robotics research platform that remains freely accessible to anyone.

  • OFFWORLD GYM - Open-access physical robotics environment for real-world reinforcement learning benchmarking and research.

Eletronics

  • https://www.falstad.com/circuit/ - Online analog circuit simulator

  • simcirjs - Digital circuit simulator in HTML5 and Javascript

  • Yosys2digitaljs - Online SystemVerilog synthetizer

Miscellaneous

  • nbinteract - Generate Interactive Web Pages From Jupyter Notebooks. See also this tutorial.

  • RISE - Turn Jupyter notebooks into slideshow

  • MathBox - Presentation-quality WebGL math graphing (see also How to use Jupyter with MathBox2 plots)

  • https://www.eulertour.com/ - Online editor for Manim, an animation engine for explanatory math videos

  • https://cindyjs.org/ - A framework to create interactive (mathematical) content for the web.

  • SageMathCell - SageMathCell can be used to embed Sage computations into any webpage.

  • Wolfram Demonstrations Project - Database of interactive demonstrations made with Mathematica.

  • Grader - Automatically grade MATLAB code in any learning environment

  • For recording online courses

    • https://www.descript.com/ - Edit audio by editing text. (English only)

    • https://www.videopuppet.com/ - Use markdown syntax to create video clips from images, add captions, and life-like narration audio from your text. (Has Portuguese narration)
